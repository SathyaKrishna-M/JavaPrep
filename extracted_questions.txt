5. Explain about different selectors in CSS.
CSS selectors are patterns used to select HTML elements for applying styles.

--- PAGE BREAK ---

1. Tag (Element) Selector
Selects all elements of a specific tag.
Example:
p { color: blue; }
2. Class Selector
Selects elements with a specific class name.
Example:
.box { padding: 10px; }
3. ID Selector
Selects an element with a unique ID.
Example:
#header { background: black; }
4. Universal Selector
Applies style to all elements.
* { margin: 0; }
5. Group Selector
Applies the same style to multiple elements.
h1, h2, h3 { font-family: Arial; }
6. Descendant (Nested) Selector
Applies style to elements inside other elements.
div p { color: green; }
7. Attribute Selector
Selects elements with specific attributes.
input[type="text"] { border: 1px solid black; }

G. What is box model in CSS?
The CSS Box Model describes how every HTML element is represented as a rectangular box. It
defines the space an element occupies and consists of four layers:
1. Content – Actual text or image inside the element.
2. Padding – Space between content and border; adds inner spacing.
3. Border – A line surrounding the padding and content.
4. Margin – Space outside the border; creates spacing between elements.

--- PAGE BREAK ---

It helps in designing layout, alignment, and spacing of elements accurately.
10. Explain the need of border in designing phase.
The border is important in the design phase because:
 It visually separates one element from another.
 Helps in identifying sections, boxes, and containers.
 Useful for creating focus areas (ex: forms, cards, buttons).
 Helps developers debug spacing by showing an element’s actual size.
 Enhances aesthetic appearance, giving structure to layouts.
 Used to create effects like outlines, separators, and UI components.

12. Explain about margin and padding in CSS.
 Margin:
o Space outside the border.
o Creates distance between two elements.
o Does not affect the element’s background color.
o Example: margin: 20px;
 Padding:
o Space inside the border (around content).
o Increases the clickable/content area.
o Affects background color because it is inside the box.

--- PAGE BREAK ---

o Example: padding: 10px;
Both are essential for layout spacing and readability.

14. What is the purpose of HTTPS? How does it differ from HTTP?
Purpose of HTTPS:
HTTPS (HyperText Transfer Protocol Secure) is used to secure communication between
browser and server. It protects data using encryption.
Difference:
HTTP HTTPS
Not secure Secure, encrypted
Data can be intercepted Data is protected using SSL/TLS
Uses port 80 Uses port 443
No certificate Requires SSL certificate
HTTPS ensures privacy, integrity, and authenticity.
15. What are web protocols? Name any two and explain briefly.
Web protocols are rules that define how data is transmitted over the internet.
Two important protocols:
1. HTTP/HTTPS – Used for transferring webpages between client and server. HTTPS adds
encryption.
2. FTP – Used for uploading and downloading files between client and server.

--- PAGE BREAK ---

Other examples: SMTP, IMAP, TCP, UDP.
16. Explain the client–server architecture used in the web.
Client–server architecture is a model where:
 The client (browser) sends a request.
 The server processes the request, retrieves data, and sends a response.
Steps:
1. User enters a URL.
2. Browser sends HTTP/HTTPS request to server.
3. Server processes the request.
4. Server sends back HTML/CSS/JS files.
5. Browser displays the webpage.
It separates UI (client) from data processing (server).
